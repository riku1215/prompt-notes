# 日本語 LLM の文体ばらつき対策

## 観察された問題

| プロバイダ | 症状 |
|---|---|
| GPT-4o | temp 0.9 + 親友 persona でため口混在 |
| Claude 3.5 Sonnet | 「〜してください」命令形を多用 |
| Gemini | 「ですわ」など過剰な丁寧表現の混入 |

## 解決策

### 1. プロンプトで「文体テンプレ」を明示

```
文体例:
- 良い例: 「お疲れさまです。今日のご様子から、少しお疲れがあるように感じます」
- 悪い例: 「お疲れさま！今日はちょっと疲れてるみたいだね」
```

### 2. 短い文末ルール集を最後に置く

```
末尾文体ルール:
- 「〜してください」(命令形) は使わない
- 「〜してみてはいかがでしょうか」「〜のもよいかもしれません」を使う
- 最後の文は相手への気遣いか具体的な行動提案で締める
```

### 3. システム vs ユーザー分離

カウンセリング系は system に persona、user に状況のみ:
- system: 文体・persona・NG リスト
- user: 直近の状況 + 観察データ
