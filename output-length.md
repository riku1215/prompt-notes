# 出力長を 300〜500 字に収める

## 問題

LLM は temp 0.9 で**冗長**に書きがち。
モバイル UI では 500 字超は読まれない。

## 効果のある指示順

```
1. 「〜字程度で」 (推奨幅指定)
2. 「全体ではなく特に気になる 2-3 点だけに焦点を絞り」 (情報量制限)
3. 「全項目を網羅的に取り上げないでください」 (明示否定)
```

## max_tokens 設定

- 日本語 1 字 ≈ 1〜1.5 token
- 500 字 → max_tokens 800 で余裕
- 短すぎると途中切れ、長すぎると無駄なコストとレイテンシ

## 検証コード

```js
const len = response.length;
if (len > 600) console.warn('too long:', len);
if (len < 200) console.warn('too short:', len);
```
