# メタ表現を抑える NG リスト方針

## 問題

LLM は応答時にしばしば**回答プロセス自体に言及**する:
- 「あなたの回答を見ていると」
- 「ご回答を拝見して」
- 「アンケートの結果から」

これらは:
1. ユーザー体験を阻害する (アンケートを受けている感が強まる)
2. 自然な会話の流れを壊す
3. 「相談相手としての AI」という設計意図と矛盾する

## NG リスト (実運用版)

```
【禁止表現】
- 「あなたの回答を見ていると」「ご回答を拝見して」「自由記述の〜」など、
  アンケート構造やメタ情報に言及する表現
- 「私も親友として」「一緒に考えます」など自分(送り手)を人物として登場させる表現
- 「〜してくださいね」「〜してみてください」等の指示・命令形
- 点数・スコア・軸名・質問番号の直接引用
- 回答内容を引用するときは
  「疲れていると感じておられるようですね」のように、
  相手の状態として自然に言及する
```

## 効果測定

- NG リスト追加前: 出力の 60% にメタ表現混入
- NG リスト追加後: メタ表現混入率 5% 以下

## 補強ガード (任意)

```js
const META_PATTERN = /(あなたの回答|ご回答を拝見|自由記述の|アンケート|質問項目)/;

  // 再生成 or 警告ログ
}
```
